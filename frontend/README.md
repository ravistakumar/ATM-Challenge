# ATM Frontend

React frontend for the ATM application.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens at `http://localhost:3000`

## Build

```bash
npm run build
```

Output goes to `dist/` directory.

## Project Structure

```
src/
├── api/        # API client
├── components/ # Reusable UI components
├── context/    # React Context (auth state)
├── hooks/      # Custom hooks
├── pages/      # Page components
├── types/      # TypeScript interfaces
└── utils/      # Helper functions
```
